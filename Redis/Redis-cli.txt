Here's a comprehensive list of Redis CLI commands organized by category:

Connection Commands:

redis-cli                    # Connect to localhost:6379
redis-cli -h host -p port    # Connect to specific host/port
redis-cli -a password        # Connect with password
redis-cli --user username -a password  # Connect with username/password
PING                         # Test connection
AUTH password                # Authenticate
SELECT index                 # Switch database (0-15)
QUIT                         # Close connection
```

## String Commands
```bash
SET key value                # Set key to value
GET key                      # Get value of key
GETSET key value            # Set new value, return old value
MSET key1 val1 key2 val2    # Set multiple keys
MGET key1 key2              # Get multiple keys
SETNX key value             # Set if key doesn't exist
SETEX key seconds value     # Set with expiration
PSETEX key milliseconds value  # Set with expiration (ms)
APPEND key value            # Append to string
STRLEN key                  # Get string length
INCR key                    # Increment by 1
INCRBY key amount           # Increment by amount
DECR key                    # Decrement by 1
DECRBY key amount           # Decrement by amount
GETRANGE key start end      # Get substring
SETRANGE key offset value   # Overwrite part of string
```

## Key Commands
```bash
KEYS pattern                # Find keys matching pattern (*, ?, [])
SCAN cursor [MATCH pattern] # Iterate keys (safer than KEYS)
EXISTS key                  # Check if key exists
DEL key [key ...]          # Delete key(s)
UNLINK key [key ...]       # Async delete (non-blocking)
TYPE key                    # Get key type
RENAME key newkey           # Rename key
RENAMENX key newkey         # Rename if newkey doesn't exist
EXPIRE key seconds          # Set expiration
EXPIREAT key timestamp      # Set expiration (Unix timestamp)
PEXPIRE key milliseconds    # Set expiration (ms)
TTL key                     # Get time to live (seconds)
PTTL key                    # Get time to live (milliseconds)
PERSIST key                 # Remove expiration
DUMP key                    # Serialize key
RESTORE key ttl value       # Deserialize key
RANDOMKEY                   # Get random key
TOUCH key [key ...]        # Update last access time
```

## List Commands
```bash
LPUSH key value [value ...] # Push to left (head)
RPUSH key value [value ...] # Push to right (tail)
LPOP key                    # Pop from left
RPOP key                    # Pop from right
LINDEX key index            # Get element by index
LLEN key                    # Get list length
LRANGE key start stop       # Get range of elements
LSET key index value        # Set element by index
LINSERT key BEFORE|AFTER pivot value  # Insert element
LREM key count value        # Remove elements
LTRIM key start stop        # Trim list to range
BLPOP key [key ...] timeout # Blocking left pop
BRPOP key [key ...] timeout # Blocking right pop
RPOPLPUSH source dest       # Pop from source, push to dest
BRPOPLPUSH source dest timeout  # Blocking version
```

## Set Commands
```bash
SADD key member [member ...] # Add members to set
SREM key member [member ...] # Remove members
SMEMBERS key                 # Get all members
SISMEMBER key member         # Check membership
SCARD key                    # Get set size
SPOP key [count]            # Remove and return random member(s)
SRANDMEMBER key [count]     # Get random member(s)
SMOVE source dest member    # Move member between sets
SINTER key [key ...]        # Intersection of sets
SINTERSTORE dest key [key ...]  # Store intersection
SUNION key [key ...]        # Union of sets
SUNIONSTORE dest key [key ...]  # Store union
SDIFF key [key ...]         # Difference of sets
SDIFFSTORE dest key [key ...]   # Store difference
SSCAN key cursor            # Iterate set members
```

## Sorted Set (ZSet) Commands
```bash
ZADD key score member [score member ...]  # Add members with scores
ZREM key member [member ...]              # Remove members
ZSCORE key member                         # Get member score
ZINCRBY key increment member              # Increment score
ZCARD key                                 # Get set size
ZCOUNT key min max                        # Count members in score range
ZRANGE key start stop [WITHSCORES]        # Get range by index
ZREVRANGE key start stop [WITHSCORES]     # Get range (reversed)
ZRANGEBYSCORE key min max [WITHSCORES]    # Get range by score
ZREVRANGEBYSCORE key max min [WITHSCORES] # Get range by score (reversed)
ZRANK key member                          # Get rank (0-based index)
ZREVRANK key member                       # Get rank (reversed)
ZPOPMIN key [count]                       # Remove lowest score(s)
ZPOPMAX key [count]                       # Remove highest score(s)
ZUNIONSTORE dest numkeys key [key ...]    # Store union
ZINTERSTORE dest numkeys key [key ...]    # Store intersection
ZSCAN key cursor                          # Iterate sorted set
```

## Common Patterns & Examples
```bash
# Pattern matching
KEYS user:*              # All keys starting with "user:"
KEYS *name*              # All keys containing "name"
KEYS user:???            # Keys like user:123

# Expire patterns
EXPIRE session:123 3600  # Expire in 1 hour
TTL session:123          # Check remaining time

# Atomic operations
INCR page:views          # Increment counter
INCRBY cart:total 50     # Add to total

# Lists as queues
LPUSH queue:jobs task1   # Add to queue
BRPOP queue:jobs 0       # Wait for and process job
```
